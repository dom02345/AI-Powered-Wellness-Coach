# 🤖 AI Code Reviewer

> 🚀 This project demonstrates the practical use of Generative AI in automating code reviews using Large Language Models (LLMs).

This project is primarily focused on **Generative AI-based code analysis**, rather than frontend design or UI complexity.

While the application includes a simple and clean web interface, the main objective is to demonstrate how **Large Language Models (LLMs)** like Gemini can be used to perform intelligent code reviews. The system uses prompt engineering techniques to analyze code logic, evaluate time and space complexity, identify issues, and generate structured, human-like feedback.

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login with JWT)
- 💻 Code Submission (Multiple programming languages)
- 🤖 AI-Powered Code Review
- 📊 Structured Feedback:
  - Approach Analysis
  - Time & Space Complexity
  - Strengths
  - Issues & Problems
  - Improvements & Optimizations
  - Final Verdict
- 🕘 Review History (stored per user)
- 🎨 Clean Black & White Modern UI
